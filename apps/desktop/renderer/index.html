Ghost Developer Studio Desktop

The bundled desktop shell is ready, but the full studio UI is served by the web app.

To enable desktop parity, set GHOST_DESKTOP_WEB_URL to a running Ghost Studio web instance or start @ghost/web locally on the default development URL http://localhost:3000.

  • Launch the web studio and reopen the desktop app.
  • Keep using the secure preload bridge for local filesystem access.
  • Use this shell as the fallback experience when the web studio is unavailable.
